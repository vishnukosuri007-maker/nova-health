# NovaHealth

A modern health and wellness tracking application designed to help you monitor and improve your daily health habits.

## Overview

NovaHealth is a comprehensive health assistant that helps you track various aspects of your wellness journey including workouts, nutrition, hydration, sleep patterns, and mental health. The app provides an intuitive interface to log daily activities and visualize your progress over time.

## Features

### Health Tracking
- **Workout Logger** - Record your exercise routines, duration, and intensity
- **Hydration Monitor** - Track daily water intake and set hydration goals
- **Symptom Tracker** - Log health symptoms and identify patterns
- **Period Tracker** - Monitor menstrual cycles and related symptoms

### Nutrition
- **Meal Planning** - Plan and organize your daily meals
- **Food Diary** - Log meals and track nutritional information
- **Recipe Collection** - Save and access your favorite healthy recipes

### Wellness
- **Mood Tracker** - Record daily mood and emotional patterns
- **Meditation Timer** - Track meditation sessions and mindfulness practices
- **Sleep Monitor** - Log sleep duration and quality

### User Experience
- **Multi-language Support** - Available in multiple languages
- **Secure Authentication** - Safe login with password recovery
- **Profile Customization** - Personalize your health dashboard
- **Data Privacy** - All data stored locally on your device

## Getting Started

### Requirements

- Flutter SDK 3.7.0 or higher
- Dart 3.9.0 or higher
- Chrome, Edge, or any modern web browser (for web version)
- Android Studio or Xcode (for mobile development)

### Installation

1. Clone or download this repository
2. Open terminal in the project directory
3. Install dependencies:
   ```
   flutter pub get
   ```
4. Generate required files:
   ```
   flutter pub run build_runner build
   ```

### Running the App

#### Web Browser
```
flutter run -d chrome
```

#### Windows Desktop
```
flutter run -d windows
```

#### Mobile (with device/emulator connected)
```
flutter run
```

## Usage

1. **First Time Setup**
   - Create an account with email and password
   - Complete your profile information
   - Set your health goals and preferences

2. **Daily Tracking**
   - Navigate to different sections from the home screen
   - Log your daily activities, meals, and health metrics
   - View summaries and track your progress

3. **Settings**
   - Customize your experience in the settings page
   - Change language preferences
   - Update profile information
   - Manage notification preferences

## Technology

Built with Flutter framework for cross-platform compatibility. Uses local storage to keep your data private and secure on your device.

## Support

For questions or issues, please check the documentation or contact support through the app settings.

## Privacy

NovaHealth stores all data locally on your device. No personal information is shared with external services without your explicit consent.
